# Puppeteer Basico

Ingresa a una página y ejecuta acciones en la misma.

## Prerrequisitos 📋

- Se necesita Node.Js para utilizar el software.

```
https://nodejs.org/es/
```

## Instalar los paquetes.

```
npm i
```

### Inicia 🚀:

```
node index.js
```

⚠️ Si falla solo vuelva a ejecutar el comando.
